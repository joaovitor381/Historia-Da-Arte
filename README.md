## Obras de Arte em React

### Como executar

- Clone o projeto para a sua máquina

```bash
git clone https://github.com/joaovitor381/Historia-da-Arte.git .
```

- Instale as dependências (lembre-se de acessar a pasta correta no terminal)

```bash
npm install
```

- Rode o projeto

```bash
npm run dev
```
